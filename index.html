Software Design and Engineering

For this enhancement I took the Contact Service I originally built in CS 320 and refactored it into a Python Flask REST API. The main goal was to move away from a single-file script and into a proper layered architecture with a repository layer and a service layer. This made the code much cleaner and easier to maintain. I also added full CRUD functionality while keeping the same strict validation rules from the original Java version.

This work shows I can take an existing project and improve its structure using better software engineering practices.
